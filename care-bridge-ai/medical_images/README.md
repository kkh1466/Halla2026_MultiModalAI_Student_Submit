## 사용 방법

1. 위 링크 클릭 → 위키미디어 페이지에서 "원본 파일 다운로드" 클릭
2. 이 폴더(medical_images/)에 저장
3. Care-Bridge AI에서 이미지 업로드 시 사용

모든 이미지는 Creative Commons 또는 Public Domain 라이선스입니다.
