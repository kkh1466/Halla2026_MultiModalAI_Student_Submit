# AI Second Brain

북마크/링크 자동 분류 및 요약 피드 웹 서비스

저장한 링크를 OpenAI가 자동으로 요약하고 분류하며, 에빙하우스 망각곡선 기반 복습 알림과 주간 뉴스레터를 제공합니다.

## 주요 기능

- **AI 자동 요약/분류**: OpenAI가 웹 페이지/유튜브 영상의 핵심을 3줄로 요약하고 카테고리 자동 분류
- **연관 질문 생성**: 자료별 "함께 고민할 질문 3가지" 자동 생성
- **지식 그래프**: 저장한 링크들 간 키워드 연결을 네트워크 맵으로 시각화
- **Spaced Repetition**: 중요 링크에 1일/7일/30일 복습 리마인더 자동 설정
- **학습 대시보드**: 주간/월간 지식 습관 통계 그래프
- **통합 분석 보고서**: 여러 링크를 선택하여 종합 인사이트 생성
- **주간 뉴스레터**: 매주 저장한 핵심 콘텐츠를 AI가 뉴스레터로 정리
- **북마크릿**: 브라우저 확장 없이 클릭 한 번으로 링크 추가
- **링크 유효성 검사**: 삭제된 URL 자동 감지 및 알림

## 기술 스택

- **Frontend/Backend**: Next.js 14 (App Router)
- **Database**: PostgreSQL + Prisma ORM
- **AI**: OpenAI API 4.0 mini
- **Auth**: NextAuth.js (Google OAuth)
- **YouTube**: YouTube Data API v3
- **Email**: Nodemailer (SMTP)
- **Styling**: Tailwind CSS

## 법적/보안 원칙

1. **서버 크롤링 대상**: 공개 페이지만 (메타태그 + 본문 텍스트)
2. **비공개 페이지**: 접근 불가 판정 시 즉시 차단, 수집 안 함
3. **원문 미저장**: AI 요약본과 메타데이터만 DB에 저장, URL 원문은 저장하지 않음
4. **사용자 동의**: 모든 등록 시 확인 과정을 거침
5. **YouTube**: Google OAuth + 공식 API만 사용
6. **할루시네이션 방지**: OpenAI 프롬프트에 "제공된 텍스트 범위 내에서만 답변" 명시

## 모바일 앱 확장

이 프로젝트는 REST API 기반으로 설계되어 있어, 별도의 모바일 앱(React Native, Flutter 등)에서 동일한 API를 호출하여 사용할 수 있습니다.

- 모든 API는 `/api/*` 경로에서 JSON으로 통신
- CORS 헤더 설정 완료
- 인증은 NextAuth 세션 또는 추후 JWT 토큰 방식으로 확장 가능
